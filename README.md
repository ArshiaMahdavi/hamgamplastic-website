# Hamgam Plastic Parsian Website

Official website and product catalog for **Hamgam Plastic Parsian**.

## 🚀 Features

- Product catalog
- Online ordering system
- Discount management
- Invoice and order handling
- Responsive design
- Docker deployment setup

## 🛠️ Technologies

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB
- Docker
- Docker Compose

## 🌐 Live Website

https://hamgamplastic.com

## 🇮🇷 معرفی پروژه

وب‌سایت رسمی شرکت **همگام پلاستیک پارسیان** که برای معرفی محصولات، مدیریت سفارش‌ها و ارائه کاتالوگ آنلاین طراحی و پیاده‌سازی شده است.

### امکانات

- نمایش محصولات
- ثبت سفارش آنلاین
- مدیریت تخفیف‌ها
- مدیریت فاکتور و سفارش‌ها
- طراحی واکنش‌گرا (Responsive)
- استقرار و مدیریت با Docker

### توضیحات

این پروژه به عنوان یک وب‌سایت واقعی برای کسب‌وکار **همگام پلاستیک پارسیان** توسعه داده شده و شامل بخش‌های فرانت‌اند، بک‌اند، مدل‌های پایگاه داده و تنظیمات استقرار می‌باشد.

هدف این پروژه ایجاد یک حضور آنلاین حرفه‌ای برای شرکت، ساده‌تر کردن فرآیند معرفی محصولات و فراهم کردن امکان ثبت سفارش برای مشتریان است.
